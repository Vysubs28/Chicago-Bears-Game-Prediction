## Project Overview
This capstone project focuses on predicting the outcomes of **Chicago Bears NFL games** using historical performance data. The objective is to analyze trends in team performance and apply machine learning techniques to forecast future results.

## Dataset
- Historical game score data from **2017 to 2025**
- Data includes team scores and opponent scores

## Methodology
- Data cleaning and preprocessing using **Pandas**
- Exploratory Data Analysis (EDA) to uncover trends
- Feature engineering to improve prediction accuracy
- Machine learning model training and evaluation
- Performance assessment using accuracy and error metrics

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn  
- Jupyter Notebook  

## Results
The project demonstrates how historical NFL data can be leveraged to make informed predictions about future game outcomes while highlighting limitations of sports forecasting models.

## Future Improvements
- Include home/away game indicators
- Add weather and injury data
- Experiment with additional ML models
- Expand analysis to other NFL teams
